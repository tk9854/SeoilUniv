클래스

- 클래스는 객체의 데이터 타입이다.
- 속성 -> 멤버 변수(필드), 행동 -> 멤버 메소드(함수) 구현

- 멤버 : 클래스 구성 요소, 중괄호{ } 안에 작성

일반 변수와 레퍼런스 변수의 차이
- 일반 변수는 변수에 데이터를 직접 저장

생성자

- 생성자는 리턴 타입을 지정할 수 없음
- 생성자의 목적은 객체 생성 시 초기화
- 생성자는 여러 개 작성 가능(생성자 중복)
	- 매개 변수의 개수와 타입이 달라야 함
- 클래스와 이름이 같음


기본 생성자
- 매개 변수가 없고, 실행 코드가 없어 아무 일도 하지 않고 단순

- 클래스에 생성자가 하나도 없을 때는 (컴파일러에 의해) 기본 생성자를 자동으로 만들어줌
	- 클래스에 생성자가 선언 되어있는 경우는 기본 생성자를 만들어주지 않음


배열은 레퍼런스
